Place your PNG screenshots in this directory with the following names so the README displays them correctly:

- console.png  -> Live operation log / Console screenshot
- analytics.png -> Analytics (donut + bar + timeline)
- data.png -> Data table (Treeview)

Example: `assets/console.png`

Tip: create screenshots by running `python main.py`, switch to the appropriate tab, and use your OS screenshot tool to save to the assets folder.
